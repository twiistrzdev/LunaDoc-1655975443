## Game
### power4
Start a Power 4 game.

<small>This command can only be executed on a server.</small>

```
£power4 [ user ]

How to use the command
£power4 @Asgarrrr
£power4 @Twiistrz
```
<small style={{ position: 'relative', top: '-15px' }}>`[]` = Required arguments, — `{}` = Optional arguments.</small>

### tictactoe
Starts a game of tic tac toe with another member.

<small>This command can only be executed on a server.</small>

```
£tictactoe [ user ] { grid size ( between 3 and 5 ) }

How to use the command
£tictactoe @Asgarrrr 4
£tictactoe @Twiistrz 5
```
<small style={{ position: 'relative', top: '-15px' }}>`[]` = Required arguments, — `{}` = Optional arguments.</small>

## Levels
### level
Operation on the exeriences of a member.

<small>This command can only be executed on a server.</small>

```
£level [ user ] [ operation ] [ quantity ]

How to use the command
£level @Asgarrrr + 100
£level all = 2
£level @role / 2
```
<small style={{ position: 'relative', top: '-15px' }}>`[]` = Required arguments, — `{}` = Optional arguments.</small>
