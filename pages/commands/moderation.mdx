## ban
Ban a user.

<small>This command can only be executed on a server.</small>

```
£ban [ user ] { reason }

How to use the command
£ban @Asgarrrr
£ban @Twiistrz Not responding LOL
```
<small style={{ position: 'relative', top: '-15px' }}>`[]` = Required arguments, — `{}` = Optional arguments.</small>

## prune
Delete a quantity of messages in a channel sent by a member, all members with a role, bots, pinned messages, files or only yours.

<small>This command can only be executed on a server.</small>

```
£prune { number } { user | role | Keyword ( me | bots | uploads | pins) }

How to use the command
£prune 100 @Asgarrrr
£prune 50 @Admin
£prune 10
£prune 100 bots
```
<small style={{ position: 'relative', top: '-15px' }}>`[]` = Required arguments, — `{}` = Optional arguments.</small>
